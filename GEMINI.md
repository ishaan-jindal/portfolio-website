# Project Overview

This is a Next.js project bootstrapped with `create-next-app`. It uses React, TypeScript, and Tailwind CSS. The project is configured with ESLint for code linting.

## Building and Running

### Development

To run the development server:

```bash
npm run dev
```

### Building

To build the project for production:

```bash
npm run build
```

### Starting the production server

To start the production server:

```bash
npm run start
```

### Linting

To run the linter:

```bash
npm run lint
```

## Development Conventions

*   **Framework:** Next.js with the App Router.
*   **Language:** TypeScript.
*   **Styling:** Tailwind CSS.
*   **Linting:** ESLint with the `eslint-config-next` configuration.
*   **Fonts:** The project uses the Geist font family.
